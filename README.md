# 🌡️ Temperature Converter Program

This project is a simple Temperature Conversion Program developed as part of **ProDigy InfoTech Internship - Task 01**.

## 📌 Project Description

The program allows users to convert temperature values between three major units:

- Celsius (°C)
- Fahrenheit (°F)
- Kelvin (K)

Users can input a temperature value along with its unit, and the program will convert it into the other two units.

---

## 🚀 Features

- Convert Celsius to Fahrenheit & Kelvin
- Convert Fahrenheit to Celsius & Kelvin
- Convert Kelvin to Celsius & Fahrenheit
- Simple and user-friendly interface (CLI-based)
- Accurate conversion formulas

---

## 🧮 Conversion Formulas

- **Celsius → Fahrenheit**  
  `(C × 9/5) + 32`

- **Celsius → Kelvin**  
  `C + 273.15`

- **Fahrenheit → Celsius**  
  `(F - 32) × 5/9`

- **Kelvin → Celsius**  
  `K - 273.15`
